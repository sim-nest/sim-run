# sim-lib-index

In one line: sim-lib-index lets the SIM command answer what exists, where it lives, and which examples prove it.

## What it gives you

It turns the public SIM Index into an everyday command surface. A reader can look
up features, packages, languages, surfaces, and examples from the installed
runtime instead of hunting through generated pages.

## Why you will be glad

The same checked graph serves people and automation, so a search result, a trace,
and an example list agree about ownership and evidence. That makes SIM easier to
inspect from a terminal, a script, or an agent session.

## Where it fits

It sits with the SIM bootloader libraries. The generated index remains the source
of truth, and this crate gives that graph a runtime face for discovery and
support workflows.
